2017 Milestones

发表于 2017-12-05 | 分类于 生活
时间 事件 收获及心得
1月末 Material-UI Demo 熟悉 ES6&React&Webpack，理解了单页面应用的开发模式
3月初 使用 Dva.js 实现 new-cross-reach 前端 新技术提升了生产力，但生产力还是得靠人
4月中 初步完成，进一步了解 saga 异步编程很神奇，Generator很强大
5月初 JWT+Apache Thrift 跑通单点登录远程调用 实际产品的场景中，写RPC还不如直接写Json API
6月中旬 使用 Docker 搭建项目环境 适用于机器数量弹性伸缩的场景，应结合容器编排工具，实际产品场景不解决问题，还会添加新问题
7月 MySQL的主从搭建和配置文件详解 结合2016下半年打下的MySQL基础，进一步了解InnoDB，也从跑数的过程中实践了学习成果
4月底至8月初 补习线性代数和概率统计 对机器学习的认识加深，技术终究还是要回归用户场景，目前唯一可能用上的就是 CTR预估，努力方向有两个，一个是设计数学模型的研究员，另一个是调神经网络参数的工程师，前一个酷炫可是对我不现实，后一个现实但不酷炫。NLP现在有许多现成的使用方便的轮子，没做太多功课
8月初 学完使用协程实现协作式多任务调度 造轮子就是酷，语言、框架、存储引擎都只是工具
9月初 C语言实现IP库的整合 熟悉C语言
10月中旬 通读Linux高性能服务器编程 熟悉Linux网络编程
11月初 NodeJS重构取数脚本 异步非阻塞是把双刃剑，控制并发是关键
11月中旬 PHP爬虫 & 源码学习 学习 Symfony 风格 & PHP source code
12月 Pig编程 了解Hadoop，开启大数据之旅
